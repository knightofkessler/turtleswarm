Module move

Module of various movement functions.

This module keeps track of the turtle's current position and maintains persistence of this position tracking across program runs. If using this module, move the turtle only with the functions provided in here, otherwise the position tracking will break.

The position is saved to a file positions.txt. Each entry contains information in the order: x y z direction. The optional fifth line includes information about the turtle's next intended move, used for avoiding race conditions between saving position to a file and random program execution cutoff.

Info:

Goes to the given position and direction, breaking blocks to get there.

  • destY int the y-coordinate of the location to go to
