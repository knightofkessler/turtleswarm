findItem (identifier, number)
Tries to select the specified item (or an empty slot) in the inventory, optionally attempting to select a slot that possesses at least a specified number of the item.

Parameters:

  • identifier string the namespaced identifier of the desired item. If identifier is empty/nil, findItem will try to find an empty slot.
  • number int (optional) the minimum number of items that the slot must contain. Defaults to 1.

Returns:

    boolean whether or not a slot with the desired item in the desired quantity exists
findLargestItemStack (identifier)
Attempts to select the largest stack of a specified resource

Parameters:

  • identifier string the namespaced identifier of the desired item.

Returns:

    int returns the size of the selected stack (which may be zero if the resource does not exit)

trashItems (trashList)

Parameters:
