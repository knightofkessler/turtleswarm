Module node

Class for the behavior tree nodes.

The task function passed into this node must take a nil or a single table as its arguments and return boolean, otherwise behavior of this class is undefined.

Info:

Node:new (task[, args=nil[, nodeType=Node.NODE_TYPE.NORMAL]])
Creates a new Node object.

Parameters:

  • task function The function to run when this node is visited in the tree traversal. This function must take a single table or nil as an argument, and it must return a boolean value of whether it successfully completed its task.
  • args table The arguments for this node's task function (default nil)
  • nodeType string The type of node this is. Can only be one of the types stored in Node.NODE_TYPE. If it isn't, it reverts to the default type. (default Node.NODE_TYPE.NORMAL)

Returns:

    Node Returns the new node object or nil if illegal arguments were given
Node:addChild (child)
Adds a child to this node. A node's children are ordered, and this adds it to the end of the children list on the right.

Parameters:

  • child Node The child node to add to this object. This function does nothing if child is nil.

Returns:

    Node the node added as a child
Node:runTree ()
Recursively traverses the tree depth-first, starting with this node as the root. Each node runs its task when visited. Conditional nodes only run their task if their previous sibling node failed to complete its task.

Returns:

    boolean whether the tree traversal succeeded and the root node's task was completed
Node:visit ()
Executes this node's task function.

Returns:

    boolean Returns whether the task function succeeded in completing its task
